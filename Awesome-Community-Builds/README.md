# Build it and share!

*There's a reason we put this on GitHub.*

**Community.**

Here you'll find awesome builds and mods from OT leaders around the community.

If you have a unique build you would like to share (and you totally should):

1. Fork this repo.
2. Create a folder in this directory with your build.
3. Send a Pull-Request (PR), and we'll feature your build.

Best of all, you'll be helping out the next person.

**Let's go!**
